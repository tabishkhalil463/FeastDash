**Backend:**
- Python 3 / Django 5.1
- Django REST Framework
- PostgreSQL
- SimpleJWT Authentication
- drf-spectacular (OpenAPI docs)

**Frontend:**
- React 18 (Vite)
- Tailwind CSS
- React Router DOM
- Axios
- React Hot Toast
